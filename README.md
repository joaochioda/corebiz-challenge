# Getting Started

## Download the project

```bash
  git clone git@github.com:joaochioda/corebiz-challenge.git
```

## Run

Before starting, make sure that port 3000 is available.

Go to project

```bash
  cd corebiz-challenge
```

Install the dependencies

```bash
  npm i
```

Run the project

```bash
  npm run start
```

## Run tests

```bash
  npm run test
```
